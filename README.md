# 📖 BUDGET APP <a name="about-project"></a>


**Hello Rails React**  
The "Hello Rails React" application is a combination of React and Rails, utilizing webpack for its build process. The primary functionality involves showcasing a random message retrieved from the Rails component of the project via an API. This message is then presented within the greetings component of the React section of the application.

## 💻 Getting Started <a name="getting-started"></a>


To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

```sh
 install ruby
 ruby --version
 install sqlite3
 sqlite3 --version
 gem install rails 
 rails --version
```


### Setup

Clone this repository to your desired folder:


```sh
  cd my-folder
  git clone https://github.com/eudondian/hello-rails-react.git
```
### Install

Install this project with:

```sh
  cd hello-rails-r-app
  bundle install
 ```
 ```sh
 raiils db:create
 ```
 ```sh
  raiils db:create
 ```
### Usage

To run the project, execute the following command:

```sh
  ./bin/dev

  http://localhost:5000/api/greetings

  ```
